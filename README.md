## AI Chatbot 🤖

An AI-powered chatbot built using Python, Streamlit and Groq API.
Ask anything and get instant intelligent responses!

## Features
- Real-time AI responses
- Chat history maintained during session
- Clean and simple chat UI
- Powered by LLaMA 3.3 70B model via Groq

## How to Run
pip install -r requirements.txt
streamlit run aichatbot.py

## Built With
- Python
- Streamlit
- Groq API
- LLaMA 3.3 70B
